# Emacs

## Emacs Basics

C = control key
M = alt/meta key

| Key stroke  | Description               |
| ----------- | ------------------------- |
| `C-x` `C-f` | Read a file into emacs    |
| `C-x` `C-s` | Save a file to disk       |
| `C-x` `C-c` | Exit emacs                |
| `C-s`       | Search forward            |
| `C-r`       | Search backwards          |
| `C-v`       | Scroll to next screen     |
| `M-v`       | Scroll to previous screen |
| `C-x` `u`   | Undo                      |
